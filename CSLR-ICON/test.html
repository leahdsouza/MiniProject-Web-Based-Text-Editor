Frequently Asked Questions

Can i open an existing file from my device on the editor text feild?

NO! At the moment, the editor is at a prototype stage. This feature will be enabled soon.

How to save the file?

On clicking save, an alert box will appear. Type the name of the file along with its extension. Our editor allows files to be save in .txt(default), .html , .java and .odt formats.

How can i find and replace?

There is no dedicated feature for find and replace. You can use Ctrl + F to find the required word or character

How to use the speech to text feature?

Click on the speech to text button and start speaking once you see the recording symbol next to the title of the web page. Earphones recommended!!!

Unable to use Admin feature?

The Admin feature has been specifically designed and authorized by the creators for maintenance of the website and for keeping tabs on the database and hence can be accessed by them only.
